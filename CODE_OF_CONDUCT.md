# Code of Conduct

## Our Pledge

Visual Assist is an accessibility-focused project designed to help visually impaired users. We are committed to providing a welcoming and inclusive environment for everyone.

## Our Standards

### Expected Behavior

- **Be Respectful** — Treat everyone with dignity and respect
- **Be Inclusive** — Consider accessibility in all discussions
- **Be Constructive** — Offer helpful feedback
- **Be Patient** — Remember that not everyone has the same experience level

### Unacceptable Behavior

- Harassment or discrimination of any kind
- Offensive comments or personal attacks
- Publishing others' private information
- Any conduct that could reasonably be considered inappropriate

## Accessibility Focus

This project exists to help visually impaired users. When contributing:

- Consider how changes affect VoiceOver users
- Test with accessibility features enabled
- Ensure all UI elements have proper accessibility labels
- Maintain clear, descriptive documentation

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainer. All complaints will be reviewed and investigated promptly and fairly.
